# 声音跑酷游戏

这是一个基于声音触发的跑酷游戏，玩家通过发出声音来控制角色跳跃，躲避障碍物。

## 游戏特点

- 通过声音控制角色跳跃
- 随机生成的障碍物
- 计分系统
- 游戏结束界面

## 安装要求

1. Python 3.7 或更高版本
2. Pygame
3. PyAudio
4. NumPy

## 安装步骤

1. 克隆或下载此仓库
2. 安装依赖包：
```bash
pip install -r requirements.txt
```

## 运行游戏

```bash
python sound_runner.py
```

## 游戏控制

- 发出声音：使角色跳跃
- ESC键：退出游戏

## 游戏规则

- 通过发出声音使角色跳跃
- 躲避红色障碍物
- 存活时间越长，得分越高
- 碰到障碍物游戏结束
